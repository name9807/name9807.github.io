flex布局

flex是加在容器上的属性,用于指定这个容器为flex布局—-此时的floatclear vertical align都不再起效

容器的属性

容器属性 作用 值
flex-direction 决定主轴方向 row /row-reverse/column/column-reverse
flex-wrap 如果一行装不下如何换行 nowrap/wrap/wrap-reverse
flex-flow 是direction和wrap的简写形式 同上
justify-ontent 项目在主轴上的对齐方式 flex-start/flex-end/center/space-between/space-around
align-items 定义在交叉轴上如何对齐 flex-start/flex-end/center/strench/baseline
align-content 定义多根轴线的对齐方式(一根主轴时该属性不起作用) flex-start/flex-end/center/strench/space-between/space-around
项目的属性 定义 值
order 项目的排列顺序,数值越小排列越靠前 默认为0
flex-grow 项目的放大比例(如果有剩余空间放大项目等比例放大) 默认为0
flex-shrink 缩小比例(如果空间不足,缩小项目等比例缩小) 默认值为1(即默认都会缩小)
flex-basis 分配多余空间之前,项目占据的主轴空间,浏览器根据该属性计算是否有剩余空间 默认值为auto项目本身大小
flex grow,shrink,basis的简写 默认值(01auto) auto(11auto) none(00auto)
align-self 允许单个项目有与其他项目不同的对齐方式,可以覆盖align-items属性
